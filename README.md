# Imperium
## Author: Rubinho
## Descripton: A Python Blockchain Project inspired in a BlockChain (Simpler)
---

## Welcome To Imperium

---

This project is meant to be an learning project, to understand how a blockchain works and how to implement it in Python. It is not meant to be used in production or to be a real blockchain.

This needs a lot of work and is still in development, but it is a good start to understand how a blockchain works and how to implement it in Python.

---

## Features:
* Show the Blockchain
* Mine a Block
